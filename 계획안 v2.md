# 달력 프린트 웹서비스 계획안 v2 (가칭)

## 1. 프로젝트 개요

- **프로젝트명**: 미정 (예: `Print Calendar`, `Paper Calendar Studio` 등)
- **기술 스택**: Next.js + React + TypeScript
- **홈페이지 목적**
  사용자가 **직접 프린트해서 쓰는 달력**을 쉽고 예쁘게 만들 수 있는 웹서비스.
  로그인 없이도 빠르게 달력을 만들고, 출력하거나 PDF로 저장할 수 있는 조용한 도구를 지향한다.
- **수익 목적**: 일정 수준의 방문자 유입 후
  **Google AdSense(구글 광고)** 를 통한 광고 수익 확보

---

## 2. 서비스 컨셉 & 핵심 가치

### 컨셉 한 줄 요약
> "책상 위에 올려둘 종이 달력을, 몇 번의 클릭으로 만들어주는 조용한 공방"

### 핵심 가치

1. **프린트 친화성**
   - 집/사무실 프린터로 바로 출력해도 레이아웃이 깨지지 않는 구조
   - 흑백 출력, 잉크 절약 모드, 여백 최적화 등 현실적인 인쇄 환경 고려

2. **간단함**
   - 3~4단계 내에 달력 생성 완료
   - 회원가입, 복잡한 계정 시스템 없이 누구나 바로 사용 가능

3. **다국가 공휴일 지원**
   - 여러 나라 공휴일을 한 장의 달력에 겹쳐 볼 수 있음
   - 해외 지사, 유학생, 글로벌 팀에 유용

4. **종이 도구 감성**
   - 화면보다 **인쇄 후 모습**이 더 중요
   - 단정한 타이포, 여백, 종이 느낌을 살린 디자인

## 3. 타깃 사용자 & 사용 시나리오

### 3.1. 타깃 사용자

- **학원/학교 담당자**
  - 학사 일정, 시험 일정, 방학 일정을 한 장에 정리해 배포하고 싶다.
- **스타트업/소규모 팀장**
  - 프로젝트 마감, 릴리즈 일정, 휴가 일정을 함께 볼 수 있는 달력이 필요하다.
- **프리랜서/개인**
  - 개인 플래너, 공부 계획, 습관 기록용 달력을 프린트해서 쓰고 싶다.
- **해외 생활자/글로벌 팀**
  - 한국 + 거주국 공휴일을 동시에 보고 싶다.

### 3.2. 주요 사용 시나리오

1. 새해가 다가올 때, **연간 달력**을 한 번에 뽑아서 벽에 붙이고 싶다.
2. 3~6개월짜리 프로젝트 타임라인을 **가로형 달력**에 정리하고 싶다.
3. 한국 + 미국 공휴일이 모두 표시된 달력을 만들어 팀과 공유하고 싶다.
4. 개인 공부 계획을 세우기 위해 **월별 달력 + 넓은 메모 공간**이 있는 양식을 만들고 싶다.

## 4. 제공 기능 (MVP 기준)

### 4.1. 달력 유형

**1차 버전(MVP)에서 제공할 달력 타입**

- **연간 달력**
  - 한 페이지에 12개월 표시
  - 벽걸이용, 팀/조직 전체 일정 조망용
- **월간 달력**
  - 1페이지 1개월 기준
  - "이번 달 일정" 프린트용
- (선택) 3개월/6개월 묶음 달력
  - 프로젝트용 중기 일정 보기 좋게

### 4.2. 기본 설정 옵션

- **기간/연도 설정**
  - 연도 선택: 예) 2025, 2026
  - 표시 기간: 전체 연도 / 특정 월 범위

- **국가별 공휴일**
  - 1차 지원 국가:
    - 한국(KR)
    - 미국(US)
    - (추가 후보) 일본(JP), 중국(CN)
  - 여러 국가를 동시에 선택할 수 있음 (예: KR + US)

- **요일/주말 설정**
  - 주 시작 요일: 월요일/일요일 선택
  - 주말 강조 여부: 회색 배경, 테두리 등

- **종이 & 방향**
  - 용지 크기: A4 / A3 (기본은 A4)
  - 방향: 세로 / 가로

- **🆕 음력 표시 (옵션)**
  - 음력 날짜 작게 병기
  - 주요 명절(설날, 추석 등) 자동 표시
  - 한국/중국/일본 시장 대응

- **🆕 주차 번호 표시 (옵션)**
  - 연간 주차 번호 (Week 1~52) 표시
  - ISO 주차 표준 지원
  - 프로젝트 관리/학사 일정용

### 4.3. 스타일 옵션 (MVP용 최소셋)

- **테마**
  - 기본 흑백 테마 (프린트 최적화)
  - 🆕 다크모드 (화면 설정용, 프린트는 흑백 유지)
- **폰트**
  - 정갈한 명조체 기반 (예: Gowun Batang 느낌)
- **날짜/메모 비율**
  - 기본형: 날짜칸 + 적당한 메모 공간
  - 메모공간 확대형은 차후 버전으로 고려

### 4.4. 간단한 메모/이벤트 입력 (MVP 포함)

- **특정 날짜에 한 줄 텍스트 입력**
  - 날짜 클릭 → 텍스트 입력 → 달력에 표시
  - LocalStorage에 저장 (새로고침해도 유지)
  - 체류 시간 증가 → 광고 노출 증가
- **기본 기능만 제공**
  - 색상/카테고리 구분은 Ver.2 이후
  - CSV 업로드는 향후 확장

### 4.5. URL 공유 기능 (MVP 포함) ⭐

- **현재 설정을 URL로 공유**
  - 예: `?year=2025&countries=KR,US&layout=yearly&paper=A4&orientation=landscape`
  - URL 파라미터로 모든 설정 인코딩
- **팀원/SNS 공유 → 바이럴 효과**
  - 트래픽 자연 증가
  - 광고 수익 기회 확대
- **구현 간단 + 효과 큼**

### 4.6. 출력 & 다운로드

- 브라우저 인쇄 기능 최적화 (`Ctrl+P` / "인쇄하기" 버튼)
- **PDF 다운로드(단일 파일)** 기능 제공
  - A4 기준 PDF
  - 사용자 입장에서는 "프린트하기"와 거의 동일한 경험
- **🆕 프린트 설정 가이드/팁**
  - 브라우저별 최적 설정 안내 (Chrome/Safari/Firefox)
  - "여백 없음" 설정법
  - A4 용지 크기 확인 알림
  - 양면 인쇄, 잉크 절약 모드 팁

## 5. 향후 확장 기능 (Ver.2 이후)

### 5.1. 달력 유형 확장

- **주간 플래너**
  - 한 주를 크게 보여주고, 하루마다 넓은 메모 공간 배치
- **근무표 / 교대 스케줄**
  - 직원별/팀별 근무일자를 빠르게 표시할 수 있는 템플릿
- **습관 추적용 달력**
  - 하루 체크박스/점찍기 형태(공부, 운동 등)
- **🆕 달력 커버 페이지**
  - 프린트 후 바인딩하는 사용자 위한 첫 페이지
  - "2025 Calendar" + 간단한 메모 공간
  - 선택적 on/off

### 5.2. 사용자 일정 입력 확장

- 일정 구분(예: 마감, 시험, 회의)에 따라 색상/아이콘 다르게 표시
- CSV/엑셀 업로드(날짜 + 제목) 기능
- 반복 일정 설정

### 5.3. 템플릿 저장 & 공유 확장

- 즐겨 사용하는 달력 설정을 템플릿으로 저장
- 나중에는 간단한 로그인/계정 기능으로 확장 가능
- 템플릿 갤러리 (커뮤니티 공유)

### 5.4. 추가 문화/절기 정보

- **24절기 표시**
  - 입춘, 하지, 동지 등
  - 농업/전통문화 관심층 타깃
- **기타 특수 기념일**
  - 국제 기념일, 종교 기념일 등

## 7. 다른 달력 사이트와의 차별점

1. **프린트에 최적화된 디자인**
   - 화려한 색보다는 "인쇄 후" 가독성 중시
   - 흑백 프린터, 저가형 프린터에서도 선명하게 나오는 레이아웃

2. **다국가 공휴일을 한 번에**
   - 한 달력에 여러 나라 휴일을 겹쳐서 볼 수 있음
   - 텍스트 색이나 표시 방식으로 구분

3. **간단한 사용 경험**
   - 로그인, 회원가입 없이 바로 사용 가능한 구조
   - "3단계 생성"에 집중:
     1) 유형 선택 → 2) 기간/국가/스타일 설정 → 3) 출력/다운로드

4. **종이 도구 감성**
   - lostvin 방향과 맞게, UI·문구·레이아웃을 "조용하고 단정한 종이도구"처럼 설계
   - 광고가 있더라도, 화면을 가득 뒤덮지 않도록 밀도 조절

5. **🆕 URL 공유로 바이럴 확산**
   - 설정을 URL로 공유 → SNS/메신저로 자연스럽게 확산
   - 다른 서비스는 대부분 로그인 필요

6. **🆕 동아시아 문화 지원**
   - 음력, 24절기, 전통 명절 표시
   - 한중일 시장 타깃

## 8. 수익 모델 (Google 광고 수익 중심)

### 8.1. 기본 전략

- **접근 장벽이 낮을수록 방문자가 늘고, 광고 수익 가능성이 커짐**
  - 회원가입 없음
  - 무료 사용
  - 공유 링크/검색 유입에 유리한 구조

### 8.2. 광고 배치 아이디어

- 홈/달력 생성 페이지의 **우측/하단**에 배너 형태로 배치
- 생성된 달력 미리보기 아래에 **지나치게 방해되지 않는 형태의 광고 영역**
- 모바일에서는:
  - 상단 고정 배너 또는 하단 작은 배너 형태

> 원칙:
> "달력을 만들기 위해 온 사용자 경험을 방해하지 않는 선에서,
> 조용히 광고를 배치한다."

### 8.3. 트래픽 확보 방향

- "연도 + 달력 + 공휴일" 검색어 대응 (예: `2026년 달력`, `2026 calendar Korea holiday`)
- 각 연도별 전용 페이지 생성 (SEO용)
- 🆕 **다국어 지원으로 해외 트래픽 대폭 확보**
  - 영어 페이지: `2026 calendar printable`, `multi-country holiday calendar`
  - 일본어: `2026年 カレンダー 印刷`
- 🆕 **URL 공유 기능으로 자연스러운 바이럴**
  - 소셜미디어 확산
  - 팀/조직 내 공유
