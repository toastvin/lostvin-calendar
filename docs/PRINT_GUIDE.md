## Chrome 프린트 가이드

### 권장 설정

1. **프린트 다이얼로그 열기**
   - 단축키: `Cmd/Ctrl + P`
   - 또는: 메뉴 → 인쇄

2. **페이지 설정**
   ```
   용지 크기: A4
   여백: 없음 (또는 최소)
   배율: 100%
   방향: 세로 (또는 설정에 따라 가로)
   ```

3. **고급 옵션**
   - ✅ **배경 그래픽** 체크 (필수!)
     - 위치: 더보기 설정 → 배경 그래픽
     - 이유: 공휴일 빨간색, 주말 파란색 표시

   - ✅ **여백** 없음 선택
     - 위치: 더보기 설정 → 여백 → 없음
     - 이유: 달력이 페이지 전체를 사용

### 스크린샷 예시

```
┌─────────────────────────────────┐
│  인쇄                            │
├─────────────────────────────────┤
│  대상: PDF로 저장 / 프린터 선택   │
│  페이지: 모두                    │
│  레이아웃: 세로                  │
│  용지 크기: A4                   │
│  페이지/장: 1                    │
│  여백: 없음          ← 중요!      │
│  배율: 기본값                    │
│                                  │
│  더보기 설정                      │
│  ├─ ✅ 배경 그래픽  ← 필수 체크! │
│  └─ ✅ 머리글/바닥글 (선택)      │
└─────────────────────────────────┘
```

### 문제 해결

**Q: 공휴일 색상이 안 나와요**
- A: "배경 그래픽" 옵션을 체크하세요.

**Q: 달력이 잘려요**
- A: 여백을 "없음"으로 설정하고 배율을 100%로 맞추세요.

**Q: PDF로 저장하고 싶어요**
- A: 대상에서 "PDF로 저장" 선택 후 저장.

## Safari 프린트 가이드

### 권장 설정

1. **프린트 다이얼로그 열기**
   - 단축키: `Cmd + P`
   - 또는: 파일 → 프린트

2. **페이지 설정**
   ```
   용지 크기: A4
   방향: 세로
   배율: 100%
   ```

3. **Safari 전용 설정**
   - ✅ **배경색 인쇄** 활성화 (필수!)
     - 위치: Safari → 환경설정 → 고급 → "웹 페이지에서 배경색 프린트" 체크
     - macOS: 시스템 환경설정 → 프린터 및 스캐너 → Safari → "배경 프린트"

### 주의사항

Safari는 기본적으로 배경색을 프린트하지 않습니다. 다음 두 가지 방법 중 하나를 사용하세요:

**방법 1: Safari 환경설정**
1. Safari 메뉴 → 환경설정 (Cmd + ,)
2. 고급 탭
3. "메뉴 막대에서 개발자용 메뉴 보기" 체크
4. 개발자용 → "배경 이미지 프린트" 체크

**방법 2: PDF 변환 후 프린트**
1. Chrome에서 PDF로 저장
2. Safari에서 PDF 열기
3. 프린트 (색상 보장)

### 문제 해결

**Q: 주말/공휴일 색상이 흑백으로 나와요**
- A: Safari 환경설정에서 "배경색 프린트" 옵션을 활성화하세요.

**Q: 여백이 너무 커요**
- A: 프린트 다이얼로그 → 세부사항 표시 → 여백 최소화.

## Firefox 프린트 가이드

### 권장 설정

1. **프린트 다이얼로그 열기**
   - 단축키: `Ctrl + P` (Windows/Linux), `Cmd + P` (Mac)
   - 또는: 메뉴 → 인쇄

2. **페이지 설정**
   ```
   용지 크기: A4 (210mm × 297mm)
   방향: 세로
   여백: 없음
   배율: 100% (축소 없음)
   ```

3. **Firefox 전용 설정**
   - ✅ **배경색 및 이미지 인쇄** 활성화 (필수!)
     - 위치: 페이지 설정 → 형식 및 옵션 → "배경색 및 이미지 인쇄" 체크

   - ✅ **페이지 크기 고정**
     - Firefox는 페이지 크기를 자동 조정할 수 있습니다
     - "페이지에 맞춤" 대신 "100%" 선택

### 스크린샷 예시

```
┌─────────────────────────────────┐
│  인쇄 미리보기                   │
├─────────────────────────────────┤
│  프린터: [선택]                  │
│  용지 크기: A4 (210 x 297mm)    │
│  방향: ⦿ 세로  ○ 가로           │
│                                  │
│  페이지 설정                     │
│  ├─ 배율: 100%                  │
│  ├─ 여백: 없음                  │
│  └─ ✅ 배경색 및 이미지 인쇄    │
│                                  │
│  [취소]          [인쇄]         │
└─────────────────────────────────┘
```

### 문제 해결

**Q: 달력이 여러 페이지로 나뉘어요**
- A: "페이지에 맞춤" 대신 배율 100% 선택.

**Q: 색상이 안 나와요**
- A: "배경색 및 이미지 인쇄" 옵션을 체크하세요.

## 공통 문제 해결

### 1. 설정 패널이 프린트되요
**원인**: 프린트 CSS가 제대로 적용되지 않음
**해결**:
- 페이지를 새로고침 (F5)
- 브라우저 캐시 삭제
- 프린트 미리보기에서 확인

### 2. 달력이 페이지를 벗어나요
**원인**: 여백 설정 또는 배율 문제
**해결**:
- 여백: "없음" 선택
- 배율: 100%로 고정
- 용지 크기: A4 확인

### 3. 공휴일 색상이 안 나와요
**원인**: 배경 그래픽 출력 설정 안 함
**해결**:
| 브라우저 | 설정 위치 |
|---------|----------|
| Chrome  | 더보기 설정 → ✅ 배경 그래픽 |
| Safari  | 환경설정 → 고급 → ✅ 배경색 프린트 |
| Firefox | 페이지 설정 → ✅ 배경색 및 이미지 인쇄 |

### 4. PDF로 저장하면 링크가 나와요
**원인**: PDF 생성 시 하이퍼링크 포함
**해결**:
- Lostvin Calendar는 링크 URL을 자동으로 숨깁니다.
- 만약 나타난다면: Chrome의 "PDF로 저장" 대신 실제 프린터 선택 → "Adobe PDF" 또는 "Microsoft Print to PDF" 사용

### 5. 모바일에서 프린트가 안 돼요
**원인**: 모바일 브라우저는 프린트 기능 제한
**해결**:
- **권장**: PDF 다운로드 버튼 사용
- 대안: "링크 복사" → 데스크톱에서 열기

### 6. 실제 프린터로 출력 시 여백이 생겨요
**원인**: 프린터 하드웨어 한계 (most printers have 5mm margin)
**해결**:
- 프린터 설정에서 "여백 없음" 또는 "무테두리 인쇄" 선택
- PDF로 저장 후 → Adobe Reader에서 "실제 크기" 인쇄

## ⚙️ 브라우저별 요약

| 항목 | Chrome | Safari | Firefox |
|------|--------|--------|---------|

## 🔧 개발자용 참고사항

### 프린트 CSS 적용 확인

```bash
# Chrome DevTools
Cmd/Ctrl + Shift + P → "Rendering" → "Emulate CSS media type" → "print"

# Firefox DevTools
Cmd/Ctrl + Shift + M → "Responsive Design Mode" → "Print" 선택

# Safari DevTools
개발자용 → "반응형 디자인 모드" → 프린트 에뮬레이션
```

### 테스트 체크리스트

- [ ] Chrome: 배경 그래픽 체크 후 프린트 미리보기
- [ ] Safari: 배경색 프린트 활성화 후 프린트 미리보기
- [ ] Firefox: 배경색 및 이미지 인쇄 체크 후 프린트 미리보기
- [ ] 실제 프린터 출력 (A4 용지)
- [ ] PDF 저장 후 열기 → 색상 확인

## 📱 모바일 프린트 가이드

### iOS Safari

1. **공유 버튼** 클릭
2. **프린트** 선택
3. 프린터 선택 또는 PDF로 저장

**제약사항**:
- iOS는 배경 그래픽 설정을 제공하지 않음
- **해결책**: "PDF 다운로드" 버튼 사용 (Week 4 기능)

### Android Chrome

1. **메뉴 (⋮)** → **공유** → **프린트**
2. 프린터 선택 또는 "PDF로 저장"

**제약사항**:
- 모바일 Chrome은 데스크톱만큼 세밀한 설정 불가
- **해결책**: "링크 복사" → 데스크톱에서 프린트

## 📞 사용자 지원

### 자주 묻는 질문 (FAQ)

**Q: 어떤 브라우저가 가장 좋나요?**
- A: Chrome을 권장합니다. 설정이 직관적이고 색상 출력이 안정적입니다.

**Q: 프린터 없이 PDF만 만들 수 있나요?**
- A: 네! "PDF 다운로드" 버튼을 사용하거나, 프린트 대화상자에서 "PDF로 저장"을 선택하세요.

**Q: 회사 프린터에서 색상이 안 나와요**
- A: 프린터 설정에서 "흑백" 대신 "컬러"를 선택하고, "배경 그래픽" 옵션을 활성화하세요.
